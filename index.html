THE IDEA

What is
Web Agent?

A web automation API by TinyFish. Give it a URL and a goal in plain English — it drives a real browser and returns structured JSON.

URL + Goal
→
Real browser
clicks, types, scrolls
→
JSON result
  • No CSS selectors, no XPath — just natural language goals
  • Works on JS-heavy, authenticated, and bot-protected sites
  • SDKs for Python (tinyfish) and TypeScript (@tiny-fish/sdk)

STEP 1

Get Your
API Key.

1. Go to agent.tinyfish.ai/api-keys
2. Sign up / log in
3. Click "Create API Key"
4. Copy & store it — shown only once!

Keys start with sk-tinyfish-. Never commit them to version control.

# Add to your shell profile (.bashrc, .zshrc)
export TINYFISH_API_KEY="sk-tinyfish-*****"
04 / 09
THE API

Three
Endpoints.

/run
Synchronous. Send request, wait, get result. Simplest option.
Quick tasks under 30s
/run-async
Fire & check. Returns a run_id immediately. Poll for result later.
Batch processing, long tasks
/run-sse
Live stream. Real-time SSE events: STARTED, PROGRESS, COMPLETE.
User-facing apps, progress UI
POST https://agent.tinyfish.ai/v1/automation/{run | run-async | run-sse}
Header: X-API-Key: $TINYFISH_API_KEY
Body:   { "url": "...", "goal": "...", "browser_profile": "lite" }

06 / 09
TIPS & TRICKS

Common
Patterns.

Live Preview.
Every run returns a streaming_url — a live browser feed you can embed in an <iframe>.
// SSE event:
{ type: "STREAMING_URL",
  streaming_url: "https://..." }

// Embed it:
<iframe src={url} />
Valid 24hrs. Read-only. Show users the "fish swimming"
Bot Protection.
Start with lite. If blocked, retry with stealth + proxy.
browser_profile: "stealth",
proxy_config: {
  enabled: true,
  country_code: "US"
}
Proxies: US, GB, CA, DE, FR, JP, AU
Human-like Goals.
Describe elements visually, not by selector. Number your steps. Add waits.
"1. Dismiss cookie banner
 2. Wait 2 seconds
 3. Click the blue Add to
    Cart button below price
 4. Extract cart total"
Visual descriptions beat CSS selectors

PYTHON SDK

Quick Start.

# pip install tinyfish from tinyfish import TinyFish client = TinyFish() # reads TINYFISH_API_KEY from env with client.agent.stream( url="https://scrapeme.live/shop", goal="Extract the first 2 product names and prices", ) as stream: for event in stream: print(event)
# Output: {'type': 'STARTED', 'run_id': 'abc123'} {'type': 'PROGRESS', 'run_id': 'abc123', 'purpose': 'Visit the page to extract product info'} {'type': 'COMPLETE', 'status': 'COMPLETED', 'result': { "products": [{"name": "Bulbasaur", "price": "$63.00"}, {"name": "Ivysaur", "price": "$87.00"}] }}

TypeScript SDK: npm install @tiny-fish/sdk — same API, same methods
